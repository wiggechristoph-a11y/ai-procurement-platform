## 🔧 **Quick Setup (5 minutes):**

### **Step 1: Get Your Claude API Key**
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Create account or login
3. Navigate to **API Keys**
4. Click **Create Key**
5. Copy your key (starts with `sk-ant-api03-...`)

### **Step 2: Create GitHub Repository**
1. Go to [GitHub.com](https://github.com) and login
2. Click **"New Repository"**
3. Name it: `procure-search-platform`
4. Set to **Public**
5. Check **"Add a README file"**
6. Click **"Create repository"**

### **Step 3: Upload Your Platform**
1. **Download** `github_platform.html` from the files I created
2. In your GitHub repo, click **"Add file" > "Upload files"**
3. **Drag and drop** `github_platform.html`
4. **Rename** it to `index.html` (important!)
5. **Commit** with message: "Add AI procurement platform"

### **Step 4: Enable GitHub Pages**
1. Go to your repo **Settings** tab
2. Scroll to **Pages** section
3. Under **Source**, select **"Deploy from a branch"**
4. Choose **"main"** branch and **"/ (root)"** folder
5. Click **Save**
6. Wait 2-5 minutes for deployment

### **Step 5: Access Your Live Platform**
Your platform will be available at:
```
https://[your-username].github.io/procure-search-platform
```
